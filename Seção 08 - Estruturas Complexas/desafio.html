Arrays com Objetos com Arrays

Exercício 03 - Informações dos Alunos

Dado um array de alunos semelhantes à:

const alunos = [ { nome: "nome 1", notas: [1,3,3,8] }, { nome: "nome 2", notas: [10,6,3,2] } ];

Mostre na tela uma lista contendo o nome e a média de cada aluno.


Resultado:

Dados do Aluno

  • Nome 1, média 3.75
  • Nome 2, média 5.25
