Exercício 3

Loop Dados

Vamos simular dois dados (1-6). Quantas vezes seria necessário jogar os dois dados até que os dois números saiam repetidos?

Algoritmo

  1. jogue um dado e guarde o número.
  2. jogue o outro dado e guarde o número.
  3. crie o contador = 0.
  4. Enquanto dado1 for diferente de dado2, some um ao contador.
  5. Jogue o dado novamente.
  6. mostre na tela qual o número se repetiu e quantas vezes os dados foram jogados.
