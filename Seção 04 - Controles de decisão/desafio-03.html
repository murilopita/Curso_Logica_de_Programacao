Desafio 03

1- Pedir para o usuário digitar dois números e mostrar a mensagem "APROVADO!" caso a média seja maior que 4 e nenhuma nota seja igual à zero.

2- Mostre a mensagem "REPROVADO!" caso a média seja menor ou igual à 4 ou exista pelo menos uma nota igual à zero.

OBS 01.: Foi pedido para o usuário digitar dois números de 0 à 10 mas o engraçadinho digitou 15. Como resolver?

OBS 02.: Foi pedido para o usuário digitar dois números de 0 à 10 mas o engraçadinho digitou números NEGATIVOS. Como resolver?
